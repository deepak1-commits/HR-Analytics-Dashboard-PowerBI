# 📊 HR Analytics Dashboard | Power BI

An interactive **HR Analytics Dashboard** built using **Microsoft Power BI** to analyze workforce data and provide actionable insights into employee attrition, demographics, job satisfaction, and overall HR performance.

---

# 📌 Project Overview

Organizations generate large volumes of HR data that can be transformed into meaningful insights for better decision-making.

This project focuses on building an interactive Power BI dashboard that helps HR professionals monitor key workforce metrics, identify attrition trends, analyze employee demographics, and support strategic HR planning through data visualization.

---

# 🎯 Objectives

* Monitor employee attrition
* Analyze workforce demographics
* Track HR performance metrics
* Identify factors affecting employee retention
* Build an interactive and user-friendly dashboard

# 🔄 Project Workflow

1. Import HR dataset
2. Clean and transform data using Power Query
3. Create relationships between tables
4. Build calculated columns and measures using DAX
5. Design interactive dashboard
6. Create KPIs and visualizations
7. Generate business insights

---

# 💡 Business Insights

The dashboard helps answer questions such as:

* Which departments have the highest attrition?
* Which age groups are most likely to leave?
* How does job satisfaction affect employee retention?
* What is the gender distribution across departments?
* Which job roles experience the highest turnover?
* How do salary and experience relate to attrition?
